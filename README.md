# Portfolio Développeur Créatif & Épuré

Ce portfolio est conçu avec Tailwind CSS et AOS.js pour un style minimaliste, moderne et animé.

## Structure
- Hero centré
- Grille de projets animée
- Timeline expérience
- Formulaire de contact épuré
- Footer minimaliste

## Technologies
- [Tailwind CSS](https://tailwindcss.com/)
- [AOS.js](https://michalsnik.github.io/aos/)

## Lancer le projet
Ouvrez simplement `index.html` dans votre navigateur.

## Personnalisation
Remplacez les sections par vos propres informations et projets.

---
Palette recommandée : blanc, gris clair, accent pastel.

---
> Ce template est prêt à l’emploi et personnalisable.
